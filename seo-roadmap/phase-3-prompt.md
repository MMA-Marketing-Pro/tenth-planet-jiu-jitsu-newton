## Schema notes for this phase

- **Curation pages:** Use `ItemList` with `position` fields — Google shows rich carousel for "best of" queries when this is set up properly.
- **Comparison pages:** Use `Article` with a comparison `Table` HTML block. Consider adding a `FAQPage` block at the bottom with 3-4 common decision questions.

## Uniqueness guard

Best-of pages must NOT be thinly disguised about-us pages. Each best-of page should legitimately discuss 3-5 alternatives in the market (even if the academy is listed first), cite honest trade-offs, and link to competitor GBP listings. Authoritative tone, not self-promotional.
