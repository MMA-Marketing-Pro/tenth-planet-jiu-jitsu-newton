# Team / Instructor Photos

Drop instructor and staff headshots here.

**File naming:** `firstname-lastname.jpg` (lowercase, hyphens — e.g. `eddie-smith.jpg`)
**Recommended:** Square crop, minimum 800×800px, dark/moody lighting that matches the cinematic-fight site feel.

For this site, the priority photos are the head coach and any featured instructors. When real photos are dropped in, update the `data-local` paths in the HTML to point here (they're already set up on every `<img>` tag).
