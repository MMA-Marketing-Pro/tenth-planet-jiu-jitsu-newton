# Logo

Drop the 10th Planet Jiu-Jitsu Newton logo files here.

**Recommended:**
- `logo.png` (or `.svg`) — transparent background, minimum 400px wide, used in navbar + footer
- `logo-light.png` — version optimized for dark backgrounds (the whole site is dark — this is the primary)
- `logo-wordmark.png` — wordmark-only variant for tight nav spaces
- `favicon.png` (32x32 and 192x192) — site favicon / PWA icon

After dropping files in here, update the `.nav-brand__glyph` block in `styles.css` or replace the inline SVG-style planet mark in the HTML with an `<img src="images/logo/logo.png">`.
